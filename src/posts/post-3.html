Introduzione a Visual Studio Code

INTRODUZIONE

Gli sviluppatori hanno due scelte per quanto riguarda il loro ambiente di lavoro, possono scegliere tra un editor di codice e un IDEs (un ambiente di sviluppo integrato in lingua inglese integrated development environment ), in informatica, è un software che, in fase di programmazione, supporta i programmatori nello sviluppo del codice sorgente di un programma.

Tra editor di codice e IDEs c’è una vasta scelta tanto per nominarne qualcuno XCode, Brackets, Sublime Text, Atom, vim, PHPStorm. La differenza tra un editor e un IDE sta nella funzionalità e nella complessità.

Per quanto mi riguarda non conosco per diretto utilizzo molti di questi software, spero in futuro poter fare un giusto paragone tra gli uni e gli altri. Ora io ho avuto l’opportunità di provare Sublime Text e VSCODE, ho iniziato a programmare circa 2 anni fa e cercando l’editor giusto sono stata spinta verso questi 2, ma alla fine ho scelto VSCODE.

PERCHE’ VSCODE?

Quando ho iniziato ero completamente persa e non sapevo come muovermi e la scelta è caduta su VSCODE per il grosso numero di sviluppatori che ne cantavano le lodi. Sono stata attratta al grosso supporto online.

Oltre le motivazioni personali questo editor ha molte ragioni che lo rendono una valida scelta tra gli editor di codice.

Questo editor è il risultato di anni e anni di esperienza nello sviluppo di editor di codice da parte di Microsoft.

L'editor è Open-Source ed è completamente gratis. E' cross platform e funziona con Mac, Linux e Windows. E' fatto con Node.js e puoi fare uso del supporto dedicato a javascript (come l’IntelliSense ed il debugging per citarne alcuni).

Ha vinto l’entusiasmo della community, ha migliaia di estensioni alcune ufficiali altre fatte dalla community e questo si è tradotto nella classifica di gradimento

Microsoft rilascia degli update ogni mese. I continui upgrade dimostrano che Microsoft ascolta i propri users, mantenendo la piattaforma sempre stabile.

INIZIAMO!

Per scaricare il programma bisogna visitare la pagina di Visual Studio Code.

L'installazione dovrebbe variare tra i sistemi operativi, quindi seguite le istruzioni per il vostro.

Quando aprite il vostro editor per la prima volta vedrete la schermata di benvenuto:

Abbiamo una toolbar sulla sinistra con 5 icone. Queste si riferiscono a:

  • The File Explorer
  • Search
  • Source Control
  • The Debugger
  • The Extensions

Ora in questo tutorial andrò a toccare solo alcune di queste icone.

Explorer

Premi il tasto “Open Folder” nella sidebar o il link che avete a disposizione nella pagina di benvenuto. Entrambi ti porteranno alla finestra per selezionare i file.

Scegli una cartella dove hai dei file di codice o di semplice testo e aprilo.

VSCode ti mostrerà il contenuto di quella cartella:

Se selezionate uno dei file, il contenuto di esso vi apparirà nel pannello principale (sulla destra).
